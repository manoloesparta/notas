La funcion de la capa de red es hacer una conexion de un lugar a otro, asignando ips

Unas de las caracteristicas mas importantes son, es un protocolo sin conexion, es poco fiable y los paquetes IP no se identifican como parte de una secuencia o pertenecen a un determinado trabajo

Mascaras de red y aplicación.
Estandares:
255.255.255.0 Clase C = Local
255.255.0.0 Clase A = Internet
255.0.0.0 Clase B = Gobierno

IP: 4 octetos, 32 bits.
De 0 a 254 + 1-Para red + 1-Para Broadcast = 255 (11111111)

Aritmética binaria:
Operación AND
0 + 0 = 0
0 + 1 = 0
1 + 1 = 1

Es una trama de 32 bit que se separan en cuatro grupos de 8 bits, estos nos permite definir las redes y sus hosts
