El sistema operativo mas usado en la actualidad para seguridad informatica, es Kali Linux, un OS basado en Debian que lo unico que tiene diferente son los repositorios de los programas o paquetes.

La lista de repositorios se encuentra en el archivo /etc/apt/source.list
En /etc encontramos los archivos de configuración.
Para encontrar todos los porgramas estan en /usr/share/
Los directorios de opciones estan en /opt/

Podemos encontrar los programas en /usr/share/ y podemos encontrar los archivos de configuracion de esos programas en /etc/

Configurar interfaces de red

<interfaz> = eth0

Usamos el comando 'ifconfig' para que nos enseñe las interfacez de redes que tenemos, generalmente la principal en Linux sera eth0

Para poder cambiar nuestro mac address usamos el comando 'macchanger' la cual podemos usar diferentes banderas como:
	-a o -A pone otro mac address
	-m pone un mac address que tu eliges
	-r pone otro mac address completamente aleatoria
Y para usarlo es 'macchanger -r eth0'

Para cambiar una mac address primero tenemos que apagar nuestra interfaz con el comando 'ifconfig eth0 down'
Y para encender
'ifconfig eth0 up'

Para encontrar otras ips conectadas a la red podemos usar el comando 'netdiscover' tiene varias banderas pero la mas importante es -i que define que interfaz va escanear la red, para usar este comando la interfaz tiene que estar encendida
'netdiscover -i eth0'
Sin ningun parametro es un DDOS en tu propiaa red

Tambien podemos usar el comando 'arpscan' que es mas rapida que netdiscover, se usa una bandera -l para decir que escane la red local y la -i para definir la interfaz
'arpscan -l -i eth0'

Para iniciar/detener cualquier servicio utilizamos el comando service el servicio y start o stop
'service apache2 start'

Para saber que servicios estamos corriendo solo usamos el comando
'netstat -ntpl'

Para encontrar que servicios podemos utilizar se encuentran en /etc/init.d

Y para saber si un servicio esta especifamente corriendo usamos service con status
'service apache2 status'
